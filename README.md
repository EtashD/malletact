# malletact
Open-source haptic music wearable for deaf and hard-of-hearing people. Replication cost ~$50.

**Replication cost: ~$50 (Amazon) — vs. $700–$1,500 commercial equivalents.**

A vest + wrist band wearable that converts ambient sound into real-time vibration across 4 motors. Designed by a percussionist. Built with the community. Free for anyone to replicate.
